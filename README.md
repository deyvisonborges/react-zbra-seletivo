## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/deyvisonborges/react-zbra-seletivo

# Entry in folder
cd react-zbra-seletivo

# Install deps with npm
npm install

# Launch the app with npm or yarn
npm run dev 

# Run tests
npm run test
```
